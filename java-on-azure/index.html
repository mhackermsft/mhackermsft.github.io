Java on Azure

Mike Hacker May 13, 2022 Training

Did you know Microsoft has a dedicated team of engineers who ensue that Java is a first class citizen on Azure?  On this page you will find a selection of resources to help you modernize and migrate your Java applications to Azure.

Java to Azure Migration Strategy

Azure Application and Code Assessment Toolkit (AppCAT for Java) - Azure AppCAT is a tool to assess Java applications (binaries) and source code to identify replatforming and migration opportunities for Azure. It helps you modernize and replatform large-scale Java applications through a broad range of transformations, use cases, and code patterns.
