Webinar | Nov 11 | Implement API Management and Develop Event-Based Solutions

Mike Hacker October 21, 2021 Events

Thursday, November 11, 2021 | 11:00 AM – 12:30 PM | (GMT-08:00) Pacific Time (US & Canada)

In complex cloud environments, you’ll want to connect to and consume both Azure services and third-party services, including responding to events from multiple sources. In this session, we’ll cover key aspects of Azure integration. You’ll see how to how to publish APIs, create policies to manage information shared through the API, and to manage access to their APIs by using the Azure API Management service. We’ll also go into details on how to build applications with event-based architectures.
