Microsoft Azure Immersion Workshop: Modernize Java Apps
Wednesday, August 17, 2022 | 10:00 AM - 3:45 PM | (GMT-08:00) Pacific Time (US & Canada)

Join Microsoft and a specialized partner for a comprehensive overview of the range of hosting options for Java apps on Azure at Microsoft Azure Immersion Workshop: Modernize Java Apps. You’ll learn how to design solutions using DevOps tooling and continuous integration and continuous delivery (CI/CD), secrets management, virtual networks, authentication and authorization, monitoring, and autoscaling.
