On November 30th the Microsoft Azure Dev Tools team announced the preview of a new Azure Load Testing service. Azure Load Testing is a fully managed Azure service that enables developers and testers to generate high-scale load with custom Apache JMeter scripts.  That's right, you can use your existing experience and knowledge with one of the most popular open-source load testing tools with Azure Load Testing.

Get started by first reading the [announcement post,](https://azure.microsoft.com/en-us/blog/introducing-azure-load-testing-optimize-app-performance-at-scale/)  check out the [preview documentation](https://docs.microsoft.com/en-us/azure/load-testing/?wt.mc_id=loadtesting_ablogannounce1_blog_azuremktg) and then c[reate your first load test](https://docs.microsoft.com/en-us/azure/load-testing/quickstart-create-and-run-load-test)!
