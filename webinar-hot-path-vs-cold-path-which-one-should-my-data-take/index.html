Webinar: Hot Path vs. Cold Path: Which One Should My Data Take?

March 24 | 2pm - 3pm Eastern | Hosted by atmosera

“Hot Path” and “Cold Path” data are buzzwords in data engineering. While the concepts themselves seem simple, it’s not always clear when one might want to use one or the other or even what constitutes one of the others. In the webinar, we’ll look at both of these and the supporting services on Azure that can make the Hot Path and Cold Path make sense. We’ll look at services like Event Hubs, Service Bus, Data Lakes, Data Factory, Stream Analytics, and others that support these workloads.
